# Instabot Spammer

An automation flooding framework designed for processing target interactions on Instagram direct message links. Built cleanly with a decoupled execution architecture, the software packages functional options to run under standard text configuration commands or a dedicated desktop visual control matrix.

## Key Features
* **Adaptive GUI Dashboard:** Modern styling implementing dark mode color styling rules containing interactive log streams and engine state controls.
* **Graceful Worker Destruction:** Implements an instant engine-halt switch (`STOP SPAMMER`) using thread-safe events to clear out background components without memory leaks.
* **Keystroke Mimic Pacing:** Breaks down automated text insertions character-by-character to satisfy event triggers of dynamic target pages.
* **Fallbacks:** Automatically tracks interface component availability and switches to terminal handling if UI wrappers encounter runtime conflicts.
